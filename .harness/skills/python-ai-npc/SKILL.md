# AI NPC 开发模式 (Python)

## 决策树

每个 NPC 的行为由优先级决策树驱动，从高到低依次判断条件并执行。

## 信任系统

NPC 之间维护信任分数 (0-100)，基于观察更新：
- 被指控 -15, 被辩护 +10, 观察到矛盾 -20
- 信任随时间缓慢回归均值(50)

## 行为状态

NPC 有日常循环：睡觉→巡逻→聚集→投票。受游戏阶段影响。

## 策略AI

狼人AI需考虑：目标价值（高信任村民优先）、嫁祸策略、不在场证明构建
